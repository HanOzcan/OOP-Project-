# RandevuPlus Hastane Yönetim Sistemi
### Projenin en güncel ve çalışan haline ulaşmak için "tunahan-dev" branchindeki kodu çalıştırınız.

## Proje Hakkında
Bu proje Java kullanılarak geliştirilmiş basit bir hastane yönetim ve randevu sistemidir.

Sistemde:
- Hasta
- Doktor
- Admin

kullanıcıları bulunmaktadır.

Kullanıcılar sisteme giriş yapabilir, randevu işlemleri gerçekleştirebilir ve bilgilerini görüntüleyebilir.

---

# Kullanılan Teknolojiler
- Java
- Java Swing (GUI)
- Gson
- JSON

---

# Proje Özellikleri

## Hasta
- Giriş yapabilir
- Doktorları görüntüleyebilir
- Randevu oluşturabilir

## Doktor
- Giriş yapabilir
- Randevularını görüntüleyebilir

## Admin
- Kullanıcı yönetimi yapabilir
- Sistem verilerini görüntüleyebilir

---

# Kullanılan OOP Yapıları
Projede aşağıdaki OOP kavramları kullanılmıştır:

- Inheritance
- Abstract Class
- Polymorphism
- Encapsulation
- Interface
- Exception Handling

Ayrıca:
- Singleton Pattern
- Factory Pattern

kullanılmıştır.

---

# Veri Saklama
Veriler JSON dosyalarında saklanmaktadır.

Gson kütüphanesi kullanılarak:
- veri okuma
- veri yazma

işlemleri yapılmıştır.

---

# GUI
Projede Java Swing kullanılarak:
- LoginGUI
- PatientGUI
- DoctorGUI
- AdminGUI

arayüzleri geliştirilmiştir.

---

# Projeyi Çalıştırma
1. Projeyi Eclipse’e import edin.
2. Gson kütüphanesini ekleyin.
3. Main.java dosyasını çalıştırın.

---


# Amaç
Bu proje nesne yönelimli programlama mantığını geliştirmek ve temel bir hastane otomasyon sistemi oluşturmak amacıyla yapılmıştır.
# Not
Projede Gson ile ilgili bir hata oluşması durumunda önce Main.java dosyasını çalıştırıp ardından programı tekrar çalıştırınız.
